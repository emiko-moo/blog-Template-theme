# コーポレートサイト向け WordPressオリジナルテーマ（軽量・保守性重視）

実務におけるクライアントワークを想定し、派手な装飾よりも「運用のしやすさ」「堅牢性」「最新のセキュリティ」に特化して開発した、コーポレートサイト用のベーステンプレートです。

## 📁 リポジトリの構成
- `[ここに自作テーマのフォルダ名を入れる]/` ：自作したWordPressテーマのソースコード一式です。
- `README.md` ：本説明書です。

## 💡 本テーマの開発目的と実務への配慮
一般的な多機能テーマはソースコードが肥大化し、表示速度の低下や脆弱性のリスクを抱えがちです。
本テーマは、中小企業のWebサイトで最も要望の多い「会社概要（テーブル組）」と「お知らせ機能」に絞り込み、またセキュリティ面からお問合せフォームを自作プラグインとしました。

## 🛠 テクニカルアピール（採用担当者様へ）
見た目は極めてシンプルですが、裏側のコードは実務レベルの品質と保守性を意識しています。

1. **最新の「Secure Custom Fields (新SCF)」への対応**
   - 開発終了した旧SCFではなく、最新の仕様とセキュリティに対応した『Secure Custom Fields』を採用しています。
   - 企業の担当者がHTMLやブロックエディタを触らず、管理画面の専用フィールドに「画像」と「テキスト」を入力するだけで安全にページを更新できる仕組みを独自に設計しました。
2. **レスポンシブ・テーブル実装**
   - 会社概要ページ等で使用する `<table>` タグは、スマートフォン等の小型デバイスで閲覧した際にも可読性を損なわないよう、丁寧なCSS（レスポンシブ対応）を施しています。

## ⚙️ 使用技術スタック
- PHP / WordPress (テーマユニットテスト準拠のテンプレート階層意識)
- HTML5 / CSS3 
- プラグイン: Secure Custom Fields (新SCF)
